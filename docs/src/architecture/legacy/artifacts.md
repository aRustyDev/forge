#### Resume Formatting
- As long as you have a simple clean format, the content is what matters
- single-column layout
- standard font (Arial, Calibri, or Computer Modern for LaTeX)
- clear section headers
- content-first design
- The sb2nov LaTeX template and Google Docs remain the most recommended formatting tools for senior tech professionals. 
- A one-page or tight two-page document with the XYZ formula ("Accomplished X as measured by Y by doing Z") for each bullet point outperforms any elaborately designed template.
